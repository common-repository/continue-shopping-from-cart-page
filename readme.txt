Add a customizable button on the WooCommerce cart page to continue shopping. Easily modify the button and label text via the admin panel.




== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the 'Setting' option and then click on the 'Continue Shopping From Cart'.
